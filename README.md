<h1 style = "color:#D70E5A;" > Parking Control</h1>

## Status - in development ⚠️

Project developed in Spring boot with Postgres to manage parking spaces in residential or commercial condominiums.

Spring dependencies:

* Validation
* Postgreesql
* Data Jpa
* Web Starter
